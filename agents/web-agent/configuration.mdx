## Setup Process

### Step 1: Basic Configuration

1. **Access Settings**
   - Log in to your Ravan.ai dashboard
   - Navigate to Agents > Web Agent
   - Click "Configure"

2. **Choose Mode**
   - Select interaction modes (chat, voice, video)
   - Set default mode
   - Configure mode switching options

3. **Appearance**
   - Upload avatar or choose from library
   - Set brand colors
   - Configure widget position

### Step 2: Advanced Settings

1. **Personality**
   - Set tone and style
   - Configure expertise level
   - Define response patterns

2. **Integration**
   - Connect to your CRM
   - Set up webhooks
   - Configure API endpoints

3. **Behavior**
   - Set greeting messages
   - Configure auto-responses
   - Define escalation rules

## Code Implementation

Add the following code to your website to implement the web agent:

```html
<!-- Ravan.ai Web Agent -->
<script>
  window.ravanConfig = {
    agentId: 'YOUR_AGENT_ID',
    mode: 'chat', // 'chat', 'voice', or 'video'
    position: 'bottom-right',
    colors: {
      primary: '#0066FF',
      secondary: '#FFFFFF'
    }
  };
</script>
<script src="https://cdn.ravan.ai/agent.js" async></script>
```

## Configuration Options

### Mode Settings

```javascript
{
  "mode": {
    "default": "chat",
    "available": ["chat", "voice", "video"],
    "autoSwitch": true,
    "preferences": {
      "chat": {
        "typingIndicator": true,
        "messageDelay": 500
      },
      "voice": {
        "autoStart": false,
        "noiseCancellation": true
      },
      "video": {
        "resolution": "720p",
        "autoAccept": false
      }
    }
  }
}
```

### Appearance Settings

```javascript
{
  "appearance": {
    "avatar": {
      "type": "custom",
      "url": "https://your-domain.com/avatar.png"
    },
    "colors": {
      "primary": "#0066FF",
      "secondary": "#FFFFFF",
      "text": "#000000",
      "background": "#FFFFFF"
    },
    "position": {
      "horizontal": "right",
      "vertical": "bottom",
      "offset": {
        "x": 20,
        "y": 20
      }
    }
  }
}
```
