# Inbound Call Handling

This guide covers the configuration and management of inbound calls with Ravan.ai phone agents.

## Basic Configuration

### 1. Inbound Settings

```json
{
  "inbound": {
    "enabled": true,
    "maxConcurrentCalls": 20,
    "queueEnabled": true,
    "queueTimeout": 300,
    "operatingHours": {
      "start": "08:00",
      "end": "20:00",
      "days": ["monday", "tuesday", "wednesday", "thursday", "friday", "saturday"]
    }
  }
}
```

### 2. IVR Configuration

```json
{
  "ivr": {
    "enabled": true,
    "greeting": "Welcome to our support line. Please select an option.",
    "options": [
      {
        "digit": "1",
        "description": "Technical Support",
        "action": "route",
        "target": "tech_support"
      },
      {
        "digit": "2",
        "description": "Billing",
        "action": "route",
        "target": "billing"
      },
      {
        "digit": "3",
        "description": "General Inquiries",
        "action": "route",
        "target": "general"
      }
    ],
    "timeout": 10,
    "maxAttempts": 3
  }
}
```

## Call Routing

### 1. Basic Routing

```json
{
  "routing": {
    "strategy": "skill-based",
    "rules": [
      {
        "condition": "language == 'es'",
        "target": "spanish_agents"
      },
      {
        "condition": "priority == 'high'",
        "target": "senior_agents"
      },
      {
        "condition": "timeInQueue > 300",
        "action": "escalate"
      }
    ]
  }
}
```

### 2. Queue Management

```json
{
  "queue": {
    "enabled": true,
    "announcements": {
      "position": true,
      "waitTime": true,
      "frequency": 30
    },
    "holdMusic": {
      "type": "custom",
      "source": "https://example.com/music.mp3"
    },
    "callback": {
      "enabled": true,
      "threshold": 300,
      "announcement": "Would you like a callback instead of waiting?"
    }
  }
}
```
