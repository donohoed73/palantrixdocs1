## Key Sections

### 1. Agents Management

The Agents section allows you to:

- Create new agents
- Configure existing agents
- Monitor agent performance
- Manage agent settings

<Callout type="info">
  Each agent can be configured for specific use cases and channels (phone, web, or both).
</Callout>

### 2. Campaigns Dashboard

Manage your outbound and inbound campaigns:

- Create new campaigns
- Monitor campaign progress
- View success metrics
- Adjust campaign settings

### 3. Analytics Hub

Track and analyze performance:

- Conversation metrics
- Success rates
- Customer satisfaction
- ROI calculations

### 4. Settings & Configuration

Manage your account and integrations:

- Account settings
- CRM integration
- API keys
- Team management
